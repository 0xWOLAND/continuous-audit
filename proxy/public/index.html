USAspending.gov Proxy Service

This is a proxy service that forwards requests to the USAspending.gov API. It helps bypass CORS issues and provides a clean interface for accessing USAspending.gov data.

How to Use

Base URL

https://[your-domain]/api/*

All USAspending.gov API endpoints are accessible by appending them to the base URL.

Examples

Get Top-Tier Agencies

Original USAspending.gov endpoint:

https://api.usaspending.gov/api/v2/references/toptier_agencies

Using this proxy:

https://[your-domain]/api/api/v2/references/toptier_agencies

Health Check

Check if the proxy is running:

https://[your-domain]/health

Features

  • CORS-enabled for browser access
  • Automatic forwarding of all USAspending.gov API endpoints
  • Health check endpoint
  • Secure headers

Rate Limits

This proxy is subject to USAspending.gov's rate limits. Please refer to their API documentation for more details.

For more information about the available endpoints, please visit the USAspending.gov API documentation.
